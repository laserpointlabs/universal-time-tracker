Getting Started
  1. Track Time: Use the CLI or web interface to start and stop sessions for your projects.
  2. View Analytics: Go to the Analytics Dashboard to see trends, breakdowns, and AI-powered recommendations.
  3. Manage Data: Use the Database Browser to view, edit, or search your projects and sessions.
  4. Export: Download your data in JSON or CSV format from the Export Data page.
